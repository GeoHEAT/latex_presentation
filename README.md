# GeoHEAT Presentation Template

A Beamer (LaTeX) template that reproduces the official GeoHEAT PowerPoint
design: white slides, blue titles, the GeoHEAT logo on every slide, and the
EU + SERI funding logos on the title and closing slides.

## Build

Requires **XeLaTeX** and the **Carlito** font (a metric-compatible Calibri clone).

```bash
latexmk -xelatex geoheat_latex_template.tex
```
